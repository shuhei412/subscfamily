# SubscFamily - 家族向けサブスクリプション管理サービス

スマートフォンに最適化された、家族単位でサブスクリプションサービスを一元管理するWebアプリケーションです。

## 🎯 主な機能

### ダッシュボード
- **月間支出の可視化**: 全サブスクリプションの合計支出を一目で確認
- **サービス一覧**: 登録されているサブスクリプションサービスの詳細表示
- **利用者表示**: 各サービスの利用者をアバター表示
- **クイックアクション**: 新規追加、制限設定ボタン

### ファミリー管理
- **メンバー一覧**: 家族メンバーの役割（管理者、大人、ティーン、子ども）を表示
- **支払い設定**: 管理者のメインカードで一括決済
- **メンバー招待**: 家族を招待する機能

### サービス詳細
- **モーダル表示**: サブスクリプションの詳細情報をモーダルで表示
- **利用者管理**: サービスの利用者を選択・管理
- **請求情報**: 次回請求日と支払い方法を表示
- **アクション**: 共有停止、プラン変更ボタン

## 🚀 セットアップと実行

### 必要な環境
- Node.js 22.13.0 以上
- pnpm 10.26.2

### インストール
```bash
cd subscfamily
pnpm install
```

### 開発サーバーの起動
```bash
pnpm dev
```

開発サーバーは `http://localhost:5173` で起動します。

### ビルド
```bash
pnpm build
```

プロダクション用のビルドが `dist` ディレクトリに出力されます。
