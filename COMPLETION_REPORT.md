## 🚀 デプロイ方法

### ローカル開発
```bash
cd subscfamily
pnpm install
pnpm dev
```

### プロダクションビルド
```bash
pnpm build
```

ビルド後、`dist/` ディレクトリをホスティングサービスにデプロイしてください。
